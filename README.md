# AI Chatbot with NLP

##  Description
This project is a simple AI chatbot built using Python and NLTK. It can understand user input and respond to basic queries.

##  Technologies Used
- Python
- NLTK

##  Features
- Responds to greetings
- Answers basic questions
- Continuous chat until user exits

##  How to Run
1. Install Python
2. Install NLTK:
   pip install nltk
3. Run the file:
   python chatbot.py
